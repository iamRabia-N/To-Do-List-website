# TODOS - Todo List Website

TODOS is a small, simple yet beautiful todo list website. It can help you record and track your daily plan list. 

## Frameworks
- HTML
- CSS
- JQuery

## Features

- Addition of tasks
- Deletion of tasks
- Responsive
- User-Friendly
